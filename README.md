# Bar Graph Clock

A visual clock that displays the current time as a bar graph, showing hours, minutes, and seconds as proportional bars.

## Features

- Real-time clock display in HH:MM:SS format
- Bar graph visualization showing:
  - Hours (0-23)
  - Minutes (0-59)
  - Seconds (0-59)
- Smooth animations as values update
- Modern, responsive design

## Usage

Simply open `index.html` in a web browser. The clock will automatically start and update every second.

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and layout
- `script.js` - Clock logic and bar graph rendering
